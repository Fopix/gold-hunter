Подключение Gulp + WebPack
==========================

1. Открываем консоль в idea

2. Команда: npm i gulp --save-dev

3. Команда: npm init

4. Команда: npm i gulp-sass gulp-sourcemaps gulp-watch gulp-concat gulp-autoprefixer gulp-clean-css webpack-stream --save-dev

5. Команда для babel: npm i babel-loader @babel/core @babel/preset-env --save-dev

6. Закинуть в корень файл (gulpfile.js), возможно нужно будет изменить пути

7. Если есть файл (package.json), то просто (npm i)
